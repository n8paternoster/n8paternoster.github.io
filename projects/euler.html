Project Euler

Project Euler is a website dedicated to challenging those interested in mathematics by posing increasingly difficult math problems that require "more than just mathematical insights" to solve. While these problems can be solved using any means possible, a knowledge of computer programming (using any language) is required for the vast majority of problems. All of my solutions are done using C++. My solutions for the first 100 problems can be found on GitHub. For more information about the project check out their about page, or to see the list of problems check out their archives.
