Algorithms (Syllabus)

The third semester in a series of programming courses, this course utilized previously covered data structures to introduce more advanced programming algorithms and related concepts. The algorithms studied included sorting algorithms, hash tables and hashing algorithms, tree structures and related algorithms, graph structures and related algorithms, and data input/parsing. Additional topics included analyzing and ranking algorithms by time and space complexity using big O notation, the concepts of inheritance, virtual methods, and polymorphism, and more object-oriented programming. The entire course was done in C++. The final project, a culmination of everything we learned, was to create a program that found the shortest path between any two cities in the continental US (using Dijkstra's shortest-path algorithm) and visually display it.

  • Traveling the USA with Vectors

    Using the geographical hash table data from the previous project along with additional road and intersection data stored in vectors (C++ dynamic arrays), this program allows the user to create a path using roads and highways. I first define custom objects to represent a town, an intersection, and a connection. The program then reads in this data and stores town data in a hash table, and intersection and connection data in vectors. The user chooses a location to start and the program repeatedly prompts the user to choose a connecting road to travel.
